## Bất đồng bộ là gì?
Giúp chương trình không bị block khi chờ dữ liệu.

## async/await
Cú pháp giúp code bất đồng bộ dễ đọc hơn.

## Ứng dụng
- Gọi API
- Xử lý dữ liệu mạng

## Kết luận
async/await giúp code JavaScript rõ ràng và dễ bảo trì.
