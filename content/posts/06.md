JavaScript là **linh hồn của Web**.
Nếu HTML là **xương sống**, CSS là **quần áo**, thì JavaScript chính là **hành động**.

Bài này tập trung **ít lý thuyết – nhiều code**, giúp bạn **nhìn là hiểu JS đang làm gì trong trình duyệt**.

## 2. JavaScript chạy ở đâu?

![DNS](../../images/23.webp)<p align="center">
<p align="center">
  <em>Minh họa chủ đề phân biệt biểu thức và câu lệnh trong JavaScript.</em>

JavaScript chạy trong **JavaScript Engine** của trình duyệt.

| Trình duyệt   | Engine         |
| ------------- | -------------- |
| Chrome / Edge | V8             |
| Firefox       | SpiderMonkey   |
| Safari        | JavaScriptCore |

Ví dụ JS chạy trực tiếp:

```html
<script>
  console.log("JS đang chạy trong trình duyệt");
</script>
```

## 3. Nhúng JavaScript vào HTML (3 cách)

### 3.1 Viết trực tiếp (demo nhanh)

```html
<button onclick="alert('Hello JS')">Click me</button>
```

Không khuyến nghị cho dự án lớn

---

### 3.2 Dùng thẻ `<script>`

```html
<script>
  alert("Hello JavaScript");
</script>
```

---

### 3.3 Dùng file `.js` (CHUẨN)

```html
<script src="app.js"></script>
```

```javascript
// app.js
console.log("Load từ file JS");
```

 Cách dùng trong thực tế

## 4. DOM là gì? (HIỂU DOM = HẾT SỢ FRONTEND)

DOM là **cây đối tượng HTML**.

HTML:

```html
<h1 id="title">Xin chào</h1>
```

JavaScript:

```javascript
const title = document.getElementById("title");
console.log(title.innerText);
```

---

## 5. Thao tác DOM – NHIỀU VÍ DỤ

### 5.1 Thay đổi nội dung

```javascript
title.innerText = "Hello JavaScript";
```

---

### 5.2 Thay đổi HTML

```javascript
title.innerHTML = "<span style='color:red'>Hello JS</span>";
```

---

### 5.3 Thay đổi CSS

```javascript
title.style.color = "blue";
title.style.fontSize = "32px";
```

---

### 5.4 Lấy nhiều phần tử

```html
<p class="item">Item 1</p>
<p class="item">Item 2</p>
```

```javascript
const items = document.querySelectorAll(".item");
items.forEach(item => item.style.color = "green");
```

---

### 5.5 Tạo phần tử mới

```javascript
const p = document.createElement("p");
p.innerText = "Đoạn văn mới";
document.body.appendChild(p);
```

---

### 5.6 Xóa phần tử

```javascript
p.remove();
```

## 6. Event – JS phản ứng với người dùng

### 6.1 Click button

```html
<button id="btn">Click me</button>
```

```javascript
const btn = document.getElementById("btn");

btn.addEventListener("click", () => {
  alert("Bạn vừa click!");
});
```

---

### 6.2 Hover chuột

```javascript
btn.addEventListener("mouseover", () => {
  btn.style.background = "yellow";
});
```

---

### 6.3 Submit form

```html
<form id="form">
  <input type="text" id="name">
  <button>Submit</button>
</form>
```

```javascript
form.addEventListener("submit", (e) => {
  e.preventDefault();
  alert("Tên: " + name.value);
});
```

---

### 6.4 Lắng nghe bàn phím

```javascript
document.addEventListener("keydown", (e) => {
  console.log("Bạn nhấn:", e.key);
});
```

---

## 7. JavaScript chạy khi nào?

### 7.1 DOMContentLoaded

```javascript
document.addEventListener("DOMContentLoaded", () => {

```

 Tránh lỗi JS chạy khi DOM chưa sẵn sàng

## 8. JavaScript bất đồng bộ – Event Loop (code nhìn là hiểu)

### 8.1 Ví dụ cơ bản

```javascript
console.log("A");

setTimeout(() => {
  console.log("B");
}, 1000);

console.log("C");
```

Kết quả:

```
A
C
B
```

---

### 8.2 fetch API (gọi server)

```javascript
fetch("https://jsonplaceholder.typicode.com/posts/1")
  .then(res => res.json())
  .then(data => console.log(data));
```

---

### 8.3 Async / Await (dễ đọc hơn)

```javascript
async function loadData() {
  const res = await fetch("https://jsonplaceholder.typicode.com/posts/1");
  const data = await res.json();
  console.log(data);
}

loadData();
```
