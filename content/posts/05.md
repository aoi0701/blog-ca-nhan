## HTTP là gì?
HTTP là giao thức truyền dữ liệu trên web.

## Fetch API
Fetch API dùng để gửi request HTTP trong JavaScript.

## Ví dụ
Fetch API thường dùng trong ứng dụng web hiện đại.

## Kết luận
HTTP và Fetch API là kiến thức quan trọng với lập trình web.
