## Mô hình Client – Server
Server lắng nghe kết nối, client gửi yêu cầu đến server.

## TCP trong Java
Java cung cấp `ServerSocket` và `Socket` để lập trình TCP.

## Ứng dụng
- Chat
- Truyền file
- Client–Server

## Kết luận
TCP Socket là kiến thức nền tảng trong lập trình mạng Java.
